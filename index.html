The Steps to Create a New Repository in GitHub.

  1. Go to Github and log in to your account.
  2. In the upper-right corner of any page, use the  drop-down menu, and select New repository.
  3. Type a short, memorable name for your repository. For example, “hello-world".
  4. Optionally, add a description of your repository. For example, "My first repository on GitHub."
  5. Choose a repository visibility and Select Initialize this repository with a README.
  6. Click Create repository.

Congratulations! You've successfully created your first repository, and initialized it with a README file.

Commit your first change

  • In your repository's list of files, click README.md.
  • click edit,On the Edit file tab, type some information about yourself
  • Review the changes you made to the file. You'll see the new content in green.
  • At the bottom of the page, type a short, meaningful commit message that describes the change you made to the file.
  • Below the commit message fields, decide whether to add your commit to the current branch or to a new branch. If your current branch is the default branch, you should choose to create a new branch for your commit and then create a pull request.
  • Click Propose file change.

Congratulations! You have now created a repository, including a README file, and created your first commit on GitHub.
